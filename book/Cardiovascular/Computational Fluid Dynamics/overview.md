## Lesson Overview
This lesson covers topics in fluid mechanics, CFD, and applying these topics to treat cardiovascular diseases. The lesson provides background knowledge on these topics, and an exercise that engages students by reviewing computer data from cardiovascular CFD simulations.


It starts with a 20 minute powerpoint lesson that covers the aforementioned topics, followed by time dedicated on working through the computer exercise with CFD data. There is an accompanying worksheet that guides students through the process of reviewing the data and interpreting it. 

The computer exercise requires access to Windows or Mac computers, ideally with windows 10 software or newer. A free, open source software called **ParaView** is required to view the data for the exercise. The exercise files are linked on this page.
